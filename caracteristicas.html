    • Caracteristicas de JavaScript


    Es un poderoso lenguaje de programación construido para el navegador Netscape en 1995..


    1 Lenguaje Dinamico

    Los tipos de datos de las variables se definen en tiempo de ejecución, y no es necesario declarar explícitamente los tipos (tipado dinámico).


    2 Multiplataforma

    JavaScript puede ejecutarse en cualquier navegador, independientemente del sistema operativo.


    3 Funciona del lado del cliente y servidor

    Aunque históricamente se ha utilizado del lado del cliente (en el navegador), con entornos como Node.js, también se usa ampliamente en el lado del servidor..


    4 Lenguaje ligero

    JavaScript es relativamente sencillo y fácil de aprender, lo que lo hace accesible tanto para principiantes como para desarrolladores experimentados.


    5 Orientado a objetos basado en prototipos

    JavaScript es un lenguaje orientado a objetos, pero a diferencia de otros lenguajes basados en clases, se basa en prototipos para herencia.



    ¿Como funciona?


    En el navegador, JavaScript interactúa con el DOM y responde a eventos.


    Motores de ejecución: El motor de JavaScript del navegador convierte el código en instrucciones optimizadas para la máquina.

    Del lado del servidor: JavaScript puede ejecutarse en servidores con entornos como Node.js, lo que lo hace muy flexible y poderoso para aplicaciones completas.


    JavaScript responde a eventos como clics, movimientos del ratón, pulsaciones de teclado, etc.

    JavaScript es síncrono, lo que significa que ejecuta el código línea por línea en el orden en que se encuentra
